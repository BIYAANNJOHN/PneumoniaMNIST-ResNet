# PneumoniaMNIST Classification with ResNet-50

## Setup
```bash
pip install -r requirements.txt
```

## Run
```bash
python train.py
python evaluate.py
```

## Notes
- Transfer learning used: ResNet-50 pretrained on ImageNet.
- Data: PneumoniaMNIST (Kaggle).
- Key metrics: Accuracy, F1-score, Precision.
